# Bank Loan Analytics

## Project Objective

Banks receive thousands of loan applications every day.

The objective of this project is to analyze historical loan application data to:

- Understand customer demographics
- Identify factors affecting loan approval/default
- Discover business insights using SQL and Python
- Build a machine learning model to predict loan outcomes
- Develop an interactive Streamlit dashboard for stakeholders

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL (MySQL)
- Scikit-learn
- Streamlit
- Git & GitHub
